## 🚀 Featured Projects

### 🇸🇦 [ArabiQ — Arabic Q&A System](https://github.com/husseinelhaddad/arabiq)
> Python · LangChain · FastAPI · RAG · BM25 · Dense Retrieval

Arabic-aware RAG system with sentence-level chunking, hybrid retrieval (dense + BM25), and α-tuned re-ranking.
Improved **Precision@3 from 62% → 81%** on a custom 250-pair Arabic QA dataset. Deployed as a FastAPI service with **sub-1.5s end-to-end latency**.

> Python · YOLOv8 · FastAPI · Streamlit · Docker

YOLOv8 model trained on 5K+ annotated food images achieving **0.82 mAP@50**. Dockerized FastAPI + Streamlit pipeline with **30% latency reduction** for real-time inference.
